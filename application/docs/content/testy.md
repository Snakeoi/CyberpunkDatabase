# Testy Umiejętności

## Wynik Testu Umiejętności

Cecha + Umiejętność + 1k10 Atakującego
przeciw Cecha + Umiejętność + 1k10 Broniącego
lub Poziomowi Trudności (PT)

## Poziomy Trudności

| Trudność PT   | Opis                                                                                | PT |
|---------------|-------------------------------------------------------------------------------------|----|
| Łatwy         | Czynności, które większość ludzi wykona bez problemu.                               | 9  |
| Codzienny     | Czynności, które wykona większość ludzi nieposiadających specjalnego przeszkolenia. | 13 |
| Trudny        | Czynności trudne do wykonania bez wyszkolenia lub naturalnych predyspozycji.        | 15 |
| Profesjonalny | Wykonanie takiej czynności wymaga wyszkolenia.                                      | 17 |
| Heroiczny     | Zwykle wymaga gruntownego wyszkolenia.                                              | 21 |
| Niewiarygodny | Tylko najlepsi potrafią tego dokonać.                                               | 24 |
| Legendarny    | O tym ludzie długo nie zapomną.                                                     | 29 |

## Krytyczny sukces

Jeśli w Teście na 1k10 wypadnie 10, rzuć tą kością
ponownie i dodaj drugi wynik rzutu do pierwszego.
Nie rzucaj ponownie, jeśli znów wypadnie 10.

### Krytyczna porażka

Jeśli w Teście na 1k10 wypadnie 1, rzuć tą kością ponownie
i odejmij drugi wynik od sumy
Cecha + Umiejętność + wynik pierwszego rzutu.
Nie rzucaj ponownie, jeśli znów wypadnie 1.

## Kolejna próba

Jeśli Test Umiejętności nie powiódł się, nie można
podejmować kolejnej próby, chyba że z jakiegoś
powodu szansa powodzenia wzrosła.

## Umiejętności Wspierające

Za zgodą MG można wykonać jeden Test
Wspierającej Umiejętności, by dostać premię +1
do głównego Testu innej Umiejętności.

### Poświęcanie dodatkowego czasu

Otrzymujesz premię +1 do Testu Umiejętności, jeśli na jej
wykonanie poświęcisz 4 razy więcej czasu niż trzeba.

## Używanie Szczęścia

Każdy wydany punkt Szczęścia dodaje +1 do Testu Umiejętności.

## Przykładowe Modyfikatory

| Okoliczność                                          | Modyfikator |
|------------------------------------------------------|------------:|
| Słabe oświetlenie                                    |          -1 |
| Nigdy tego nie robiłeś                               |          -1 |
| Skomplikowana czynność                               |          -2 |
| Brak odpowiednich narzędzi lub części                |          -2 |
| Poprzednią noc spędziłeś w nieodpowiednich warunkach |          -2 |
| Wyjątkowo stresująca sytuacja                        |          -2 |
| Wyczerpanie                                          |          -4 |
| Bardzo pijany lub pod wpływem środków uspokajających |          -4 |
| Próba ukradkowego wykonania czynności                |          -4 |
| Ograniczona widoczność (dym, ciemność)               |          -4 |
